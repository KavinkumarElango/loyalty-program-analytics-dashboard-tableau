# Air Canada Loyalty Program Analytics Dashboard

## Overview

This project presents an interactive Tableau dashboard designed to analyze Air Canada's Loyalty Program performance. The dashboard provides insights into customer engagement, loyalty status distribution, flight activity, points accumulation, redemption behavior, and customer lifetime value (CLV) across Canadian provinces.

The objective of this project is to transform raw loyalty program data into meaningful business insights that can support customer retention strategies and data-driven decision-making.

---

## Business Problem

Airlines rely heavily on loyalty programs to retain customers and encourage repeat travel. Understanding member behavior, loyalty card performance, redemption patterns, and customer value is essential for improving engagement and maximizing program effectiveness.

This dashboard helps answer key business questions such as:

* How active are loyalty program members?
* Which loyalty cards perform best?
* What are the customer lifetime values across provinces?
* How do flight activities influence points accumulation?
* What percentage of members cancel their memberships?
* Which regions contribute the highest customer value?

## Dashboard Features

### KPI Summary

* Total Loyalty Members
* Total Flights
* Total Distance Traveled
* Total Points Accumulated
* Redemption Percentage
* Cancelled Members Percentage

### Visualizations

#### Monthly Loyalty Activity

Tracks total flights and points accumulated over time.

#### Customer Distribution by Loyalty Status

Compares Active and Cancelled members.

#### Loyalty Card vs Average CLV

Evaluates customer value across loyalty card categories.

#### Province-wise Average CLV

Geographical analysis of customer lifetime value across Canada.

#### Customer Distribution by Loyalty Card

Analyzes member distribution among different loyalty card types.

#### Comprehensive Table View

Provides detailed provincial performance metrics including:

* Average CLV
* Flights
* Members

### Interactive Filters

* Loyalty Status
* Loyalty Card
* Gender
* Province
* Year

## Key Insights

* Over 16,000 loyalty members were analyzed.
* Members accumulated nearly 800 million loyalty points.
* Active members significantly outnumber cancelled members.
* Customer lifetime value varies across provinces.
* Different loyalty card tiers demonstrate varying customer engagement levels.
